Key Features

Reporting a Lost Item

As an owner of a lost item, I want to report the lost item with detailed information, so that others can help me find it.

Report a Found Item

As a Responsible neighbor, I want to report the found items with detailed information, so that the rightful owner can reclaim it.

Search for Lost Items

As a user who has lost an item, I want to search through the list of reported found items, so that I can find and reclaim my lost item.

Claim a Found Item

As an owner of a lost item who believes a found item is mine, I want to provide proof of ownership and communicate with the Quality checker, so that I can reclaim my lost item.

About

The Lost & Found project was inspired by the need to help people find their lost items or return found items to their rightful owners. This is a semi-automated matching process where the application matches lost and found item reports, and a human quality checker verifies the match. This project was developed as a Portfolio Project for Holberton School.
